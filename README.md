# Personal Portfolio Website

## Project Overview

The **Personal Portfolio Website** is a beginner-friendly web development project created using **HTML5**. The purpose of this project is to introduce myself through a well-structured web page while demonstrating my understanding of fundamental HTML concepts.

The website contains information about my profile, educational background, technical skills, hobbies, career objectives, profile picture, and contact details. It also includes a navigation menu for easy access to different sections and a contact form with built-in HTML validation.

This project was developed as part of my internship/academic learning to strengthen my knowledge of web development fundamentals.
